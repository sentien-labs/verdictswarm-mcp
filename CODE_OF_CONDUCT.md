# Code of Conduct

VerdictSwarm MCP is for practical collaboration on agent-facing crypto risk tooling.

## Expected Behavior

- Be respectful and direct.
- Critique ideas, code, data, and assumptions rather than people.
- Explain tradeoffs when you disagree.
- Keep discussion focused on making the package safer, clearer, and easier to use.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing another person's private information.
- Spam, misleading claims, or unrelated self-promotion.
- Intentionally submitting secrets, private API data, private reports, wallet/customer research, or non-public user data.

## Reporting

If a behavior issue needs maintainer attention, contact the maintainers through this repository. For secrets, data exposure, or vulnerabilities, use private vulnerability reporting when available instead of opening a public issue.

Maintainers may remove comments, close issues, or block accounts that make the project less safe or useful for contributors.
